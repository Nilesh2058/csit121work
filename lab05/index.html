Before this, I didn’t fully appreciate just how much control CSS gives you over things like layout, colors, and fonts. One of the most helpful things I picked up was learning how to use classes and IDs to apply different styles to elements. It’s also made me think more about web accessibility and how crucial it is to have consistent styling for a better user experience. Best Design Practices

One key lesson I've gained in web design is the significance of creating responsive and accessible websites. By incorporating a single-column design for mobiles and a multi-column design for desktops, I was able to observe how a carefully constructed webpage can offer a smooth user experience on various devices. By utilizing CSS media queries and Flexbox/Grid, I was able to adjust the layout according to screen size, guaranteeing the content stays user-friendly and aesthetically pleasing. This encounter emphasized the importance of creating with the user as the central focus, placing emphasis on readability, easy navigation, and keeping the design simple. Responsive design not just improves user experience but also guarantees websites remain pertinent in a society where mobile accessibility is more crucial.

Key Learnings

  • Responsive design using media queries and grid/flexbox
  • Accessibility considerations like contrast and font size
